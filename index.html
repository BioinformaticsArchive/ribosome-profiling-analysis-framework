Ribosome profiling analysis framework

This ribosome profiling analysis framework has been described in de Klerk E., Fokkema I.F.A.C et al (2015). It features alignment, triplet periodicity analysis, detecting translation initiation sites (TISs), translation of upstream open reading frames (uORFs) and analysis of TIS motifs.

This analysis framework is maintained by @ifokkema.

When using or discussing this analysis framework please refer to: de Klerk E., Fokkema I.F.A.C. et al (2015). Assessing the translational landscape of myogenic differentiation by ribosome profiling. Nucleic Acids Research. DOI: 10.1093/nar/gkv281.
