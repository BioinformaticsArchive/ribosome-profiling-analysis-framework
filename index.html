    Ribosome profiling analysis framework

    This ribosome profiling analysis framework has been described in de Klerk E., Fokkema I.F.A.C et al (2015). It features read alignment, checking of triplet periodicity, discovering translation initiation sites (TISs), analysis of TIS motifs and translation of upstream open reading frames (uORFs).

    This analysis framework is maintained by @ifokkema.
